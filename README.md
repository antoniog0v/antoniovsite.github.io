## Project Overview:

Tasked with creating a Portfolio website that highlights my skills, experience,
projects and familiar technology, I've created a site that is aesthetically pleasing,
clean and easy to navigate for those who are interested in my achievements and accomplishments.

## Target Audience:

This site in intended for either future employers or clients to get a strong idea
of my areas of expertise, or those who are simply interested in my skillset and what
I have to offer.

## Content Strategy:

Displayed on my website are descriptions of me and my skillset, a showcasing of my portfolio, and various means of contacting me.

## Information Organization:

The website opens into a hero section describing me and my role as a Software Developer. As you scroll down, you get a small description of me that
leads the viewer into my about section. If you are to scroll back up to the navigation bar, you can easily navigate between the sections of the site
such as my skills, portfolio, and even a resume page.

## Interaction/Functionality:

The website includes several interactive elements to improve navigation and usability. A responsive navigation bar allows users to easily move between sections of the site. Buttons and links provide quick access to project repositories, professional profiles, and contact options. The contact page includes a form with JavaScript validation to ensure that users provide valid information before submitting a message. Subtle hover effects and interactive elements help guide the user while maintaining a clean and professional design.

## Technical Overview:

The portfolio website was developed using modern web development technologies including HTML, CSS, JavaScript, and the Bootstrap framework. Bootstrap was used to create a responsive layout and consistent design across all pages. Custom styling was implemented using CSS to maintain a clean and minimal visual style. JavaScript is used to support form validation and interactive elements. The project is version controlled using Git and hosted through GitHub, allowing for organized development and easy updates.
